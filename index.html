O que é HTML

(Voltar)

HTML, ou HyperText Markup Language, é a linguagem padrão utilizada para criar e projetar páginas web. Ele funciona como uma linguagem de marcação que define a estrutura e o conteúdo de uma página web, utilizando uma série de elementos e tags que são interpretados pelos navegadores da web para exibir o conteúdo de forma formatada e interativa para os usuários.

Sobre as tags

(Voltar)

As tags são um conjunto de elementos que se conectam entre si no HTML

São as tags que abrem possibilidades para a realização de um objetivo, as tags são responsáveis por criar toda a estrutura que está sendo construida e que se torne de forma visual ao usuário, variam desde titulos, textosm, imagens, links, etc.

Tags
